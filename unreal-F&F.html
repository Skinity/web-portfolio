Link: Project Link

Engine and Tools: Unreal Blueprinting, Level Editor

Time on Project: 3 weeks

Insight

Fast and the Fury is a PC racing game developed in Unreal Engine. It recreates the Monza Track, built using Unreal's default car template and then expanded.

The game tracks your current lap time, best lap, and previous lap times. The track has been enhanced with additional elements such as icy roads and a dirt shortcut for added challenge. It also offers full gamepad support.
